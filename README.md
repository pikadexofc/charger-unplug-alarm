# Charger Unplug Alarm 🔌🚨

An Android security utility built in Kotlin that triggers alert notifications the moment your charger is disconnected from the device.

## 🚀 Key Features

* **Instant Alarm Triggers**: Hardware broadcast receivers intercept power loss states.
* **Custom Alarms**: Configure loop counts, volumes, and alert ringtones.
* **Background Service**: Stays active in the system tray.
